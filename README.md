# Discord Webhook

### Requirements

1. Python >=3.9
2. venv

### Installation

```zsh
% git clone https://git.hereus.net/islekcaganmert/discord-webhook.git
% cd discord-webhook
% python3 -m venv venv
% . ./venv/bin/activate
% pip install -r requirements.txt
% echo "export export DISCORD_TOKEN={TOKEN}" > .env
% echo "{}" > db.json
```

### Run

```zsh
% python3 main.py
```
